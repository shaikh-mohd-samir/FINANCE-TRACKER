# Finance Tracker (MERN Stack)

A full-stack Finance Tracker web application built using the MERN stack.  
It allows users to manage income and expenses with secure authentication and OTP verification.

## Features
- User authentication with JWT
- OTP verification for signup & password reset
- Add and view income/expense transactions
- Protected routes (Home & Profile)
- Profile management
- Responsive UI

## Tech Stack
- Frontend: React, Tailwind CSS, Vite
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT, OTP

## Setup Instructions
1. Clone the repository
2. Install backend dependencies
3. Install frontend dependencies
4. Add environment variables
5. Run backend and frontend servers
